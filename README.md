# 📚 Book Social Platform

A scalable full-stack social media platform for book lovers built using React.js, FastAPI, Redis, MongoDB, and Docker.

---

## 🚀 Features

- User Authentication with OAuth 2.0
- Session-Based Authentication using Redis
- Create Posts, Likes, and Comments
- Secure HttpOnly Cookie Sessions
- FastAPI Async Backend APIs
- MongoDB Integration
- AI-Powered RAG Question Answering
- Dockerized Deployment

---

## 🛠️ Tech Stack

### Frontend
- React.js

### Backend
- FastAPI
- Python

### Database & Cache
- MongoDB
- Redis

### DevOps
- Docker

### Authentication & Security
- OAuth 2.0
- Session Authentication
- CSRF Protection
- Secure Cookies
- CORS Configuration

---

## 🤖 AI Feature

Implemented a Retrieval-Augmented Generation (RAG) system that retrieves relevant user-provided content and generates contextual answers using LLMs.

---

## 📌 Architecture

```text
Client (React)
      ↓
FastAPI Backend
      ↓
Redis Session Store
      ↓
MongoDB Database
      ↓
RAG Retrieval Pipeline
      ↓
LLM Response Generation
